### Завдання 2
За допомогою тернарного оператора релізувати логіку:
- є параметри `x` та `у`
- якщо `x < y` - повертаємо `x + y`
- якщо `x == y` - повертаємо `0`
- якщо `x > y` - повертаємо `x - y`
- якщо `x == 0` та `y == 0` повертаємо `"game over"`
